TO USE:
Simply create a script in Roblox Studio, paste the rblx - vrtx txt and build your map.
Run the game with the script, making sure all the parts are in workspace (can be grouped)
This will create a value in workspace named RAW_PART_DATA
Copy the Value property from it and paste into the Inspect Element console on vortex (ctrl+shift+i)
